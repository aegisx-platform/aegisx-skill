# Record Workflow Skill

Record browser workflows as animated GIFs for documentation and demonstrations.

## When to Use

- Creating tutorials or demos
- Documenting features
- Recording bug reproductions
- Making visual guides

## What It Does

1. **Starts recording** - Captures all browser actions
2. **Performs workflow** - Executes steps with pauses
3. **Takes screenshots** - Strategic frames for smoothness
4. **Exports GIF** - Downloads animated file

## Example

```
User: "Record the drug creation workflow"

Claude:
1. Starts recording
2. Navigates to drugs page
3. Clicks Add button
4. Fills form (with pauses)
5. Submits
6. Stops recording
7. Exports: drug-creation-demo.gif

✅ GIF ready for documentation!
```

## Settings

- Click indicators (orange circles)
- Action labels (descriptive text)
- Progress bar
- Quality control (1-30)
